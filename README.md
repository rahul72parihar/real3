# Knowledge Park Apartments

A single-page marketing site for Knowledge Park Apartments, North Bengaluru — a pre-launch residential project with 2, 2.5, 3 and 3.5 BHK configurations.

Static HTML/CSS/JS, no build step required.

## Structure

- `index.html` — main landing page
- `style.css` — styles
- `script.js` — nav, accordion, country-code dropdown, scroll/load animations
- `terms.html`, `privacy.html`, `disclaimer.html` — legal pages
- `assets/` — images

## Running locally

Open `index.html` directly in a browser, or serve the folder with any static file server, e.g.:

```
npx serve .
```
